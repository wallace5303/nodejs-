# 猴子选大王

### 题目

一群猴子排成一圈，按1,2,…,n依次编号。然后从第1只开始数，数到第m只,把它踢出圈，从它后面再开始数，再数到第m只，在把它踢出去…，如此不停的进行下去，直到最后只剩下一只猴子为止，那只猴子就叫做大王。要求编程模拟此过程，输入m、n, 输出最后那个大王的编号。用程序模拟该过程。


### 代码
```php
function king($n, $m)
{
    // 创建一个从 1 到 $n 的数组
    $arr = range(1, $n);

    // 从第一个开始循环
    $i = 1;

    while (count($arr) > 1) {
        // 不被整除则压入数组尾部
        if (($i % $m) !== 0) {
            array_push($arr, $arr[$i - 1]);
        }
        // 删除
        unset($arr[$i - 1]);
        $i++;
    }

    return $arr[$i - 1];
}

// 1
echo king(4, 3);
```
